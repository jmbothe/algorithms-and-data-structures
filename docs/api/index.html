algo

Namespace for algorithms.

▸ bubbleSort

Sorts an array according to a specified sort order. Worst-case time complexity is O(n^2).

Parameters
array (array) An array of values (e.g., integers, strings, objects, or anything else that can be reliably compared by the function you specify).
compare (function) Specifies a function that determines sort order. More info at https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort .
swap (boolean = true) Recursive break condition. False if previous call didn't mutate the array.
Returns
array: The sorted array.
Example
bubbleSort([2, 1, 3], (a, b) => a - b); // [1, 2, 3]
